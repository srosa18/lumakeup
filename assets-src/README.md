# Imagens — pasta de entrada

Jogue aqui as **fotos originais cruas** (qualquer tamanho/peso — pode ser export do
Figma, da câmera, etc.). Eu pego daqui, **otimizo** (resize + AVIF/WebP) e gero as
versões web em `public/images/<seção>/`. Só as versões otimizadas vão pro git.

## Onde colocar cada imagem

| Pasta aqui (`assets-src/`) | Vira (`public/images/`) | Uso |
|----------------------------|-------------------------|-----|
| `hero/`         | `hero/`         | Hero da home (e heros de página) |
| `servicos/`     | `servicos/`     | Cards de serviço (sobrancelhas, lábios, olhos, capilar, alopécia, exobrow, paramédica, reconstrução-labial) |
| `instituto/`    | `instituto/`    | Instituto Living Sculpture (retratos, mãos, materiais — nada clínico na home) |
| `lu-medical/`   | `lu-medical/`   | EverBrow / EverLashes |
| `localizacoes/` | `localizacoes/` | Fotos dos ateliês (SP, Rio, Brasília, Manaus, Miami) |
| `depoimentos/`  | `depoimentos/`  | Retratos de depoentes (quando houver direitos) |
| `fundadora/`    | `fundadora/`    | Lu Rodrigues |
| `pigmentos/`    | `pigmentos/`    | Pigmentos / materiais |
| `brand/`        | `brand/`        | Logo, monograma (de preferência **SVG**) |
| `outras/`       | —               | Não sei a seção? Joga aqui que eu classifico |

## Dicas
- **Nome ajuda mas não é obrigatório.** Se der, nomeie pelo conteúdo
  (`sobrancelhas-card.jpg`, `lu-rodrigues-retrato.jpg`). Se não, só me avise o que é.
- **Mande na maior resolução que tiver** — eu reduzo. Melhor sobrar que faltar.
- **Logo/monograma:** se tiver o **SVG** oficial, coloque em `brand/` — fica perfeito.
- Não precisa otimizar nada antes; é o meu trabalho.
